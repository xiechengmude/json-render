## 快速测试命令

### 使用 Claude Opus 测试
```bash
curl -s -X POST "http://localhost:3001/api/generate" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"构建完整的电商仪表盘：标题店铺分析的头部区域，一行4个指标(收入、订单、客户、转化率)，区域销售柱状图，以及最近订单表格","context":{"model":"opus","data":{"analytics":{"revenue":125000,"orders":567,"customers":1234,"conversionRate":0.034,"salesByRegion":[{"label":"美国","value":45000},{"label":"欧洲","value":35000},{"label":"亚洲","value":28000}],"recentTransactions":[{"id":"TXN001","customer":"艾科公司","amount":1500,"status":"completed"}]}}}}' \
  --max-time 60
```

### 使用 DeepSeek V3.2 测试
```bash
curl -s -X POST "http://localhost:3001/api/generate" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"创建销售业绩卡片，包含柱状图和2个指标","context":{"model":"ds3.2","data":{"analytics":{"revenue":125000,"growth":0.15,"salesByRegion":[{"label":"美国","value":45000},{"label":"欧洲","value":35000}]}}}}' \
  --max-time 60
```
